# Requirements

## Core Features

-   **Prompt Management:** View, search, and filter prompts.
-   **Notion Integration:** Fetch prompts from a Notion database.
-   **Responsive UI:** Mobile-first design that works on all screen sizes.
-   **Modern Design:** Clean layout with ample white space.
-   **Primary Color:** `#bd93f9`

## Technical Requirements

-   **Framework:** Vite + Vue.js
-   **UI Library:** Vuetify
-   **Backend:** Notion API for data retrieval.
-   **Environment:** Node.js and npm.

## Future Enhancements

-   Create, edit, and delete prompts directly from the dashboard.
-   User authentication.
-   Light/dark mode toggle.
