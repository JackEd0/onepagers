# Requirements

## Core Features

-   **Header:** Sticky header with brand name and navigation.
-   **Hero Section:** Background image, text, and a call-to-action button.
-   **Restaurant Menu:** A grid of 9 sushi items with images, descriptions, and prices.
-   **Testimonials:** 3 customer reviews with photos, text, and ratings.
-   **Contact Section:** Address, phone, email, social media links, and opening hours.
-   **Footer:** Copyright information.
-   **Responsive Design:** Mobile-first and fully responsive.

## Technical Requirements

-   **Framework:** Vite + Vue.js
-   **UI Library:** Vuetify
-   **Primary Color:** `#ff7f7f`
-   **Deployment:** Static site hosting (e.g., Netlify, Vercel).

## Future Enhancements

-   Online ordering system.
-   Reservation booking feature.
-   Animations and micro-interactions.
